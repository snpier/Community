##本地运行手册
    克隆代码到本地
    ```linux
        git clone https://github.com/codedrinker/community.git
    ```
    运行打包命令
    ```linux
        mvn package
    ```
    运行项目
    ```linux
        java -jar target/community-0.0.1-SNAPSHOT.jar
    ```
    访问项目
    ```html
        http://localhost:8888
    ```
